# 达尔文进化框架 Darwin Framework

让任何AI资产自主进化、只保留改进

## 一句话定义

Darwin Framework 不卖进化算法，卖AI项目的自进化能力。接入任何项目 → 自动评估→改进→验证 → 只保留有效改进。

## 核心机制

- 适者生存: 多项目矩阵自然选择
- 棘轮机制: 评估即进化，改进不可逆
- 归海机制: 自动捕获外部改进融入自身
- 进化日志: 每次改进都有记录，可追溯

## 进化成果

6项目矩阵平均提升19分，零回滚。

## 快速开始

    git clone https://github.com/MoKangMedical/darwin-framework.git
    cd darwin-framework
    pip install -r requirements.txt
    python src/darwin.py --project medichat-rd
